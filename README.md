# Movie-Recommender-system
This project is a **Movie Recommender System** built using Python and Streamlit. It suggests the top 5 similar movies based on a selected movie.

## Features
- Select a movie from the dropdown.
- Get 5 recommended movies.
- Simple and interactive UI using Streamlit.

## Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/MukeshJangid17/Movie-Recommender-system.git
cd Movie-Recommender-system
```

### 2. Create and Activate Virtual Environment
```bash
python -m venv max  # Create virtual environment named 'max'
source max/bin/activate  # For Linux/macOS
max\Scripts\activate  # For Windows
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit App
```bash
streamlit run app.py
```

## Files & Directories
- `app.py` - Main application file.
- `movie_dict.pkl` - Preprocessed movie data.
- `requirements.txt` - List of dependencies.
- `Procfile` - Configuration for deployment.
- `setup.sh` - Setup script for deployment.

## Deployment
This app can be deployed on **Heroku**, **Streamlit Cloud**, or any cloud platform supporting Python and Streamlit.
